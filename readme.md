# LAB06 – Python Object Model

## Description
This project demonstrates how Python objects work using a custom Student class.

## Setup

Create virtual environment:
python -m venv venv

Activate it:
source venv/bin/activate

Install dependencies:
pip install -r requirements.txt

## Run

python src/student.py

## Type checking

mypy --strict .

## Features

- class with attributes
- __str__ and __repr__
- equality and comparison
- sorting objects
- working with __dict__
